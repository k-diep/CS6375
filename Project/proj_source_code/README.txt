Using a Custom RNN to Predict Dallas Temperature

Dataset: https://www.kaggle.com/datasets/selfishgene/historical-hourly-weather-data

Files
- RNN.ipynb: Main code notebook. 
- log.txt: contains experiments on hyper-parameters
- RNN_text.txt: Plain text version of our code

Requirements:
- Python 3.x
- NumPy
- pandas
- Matplotlib

To run: 
Run the RNN.ipynb file and results will be printed/plotted along the way. 
Additionally, a log file will be created containing experiments on all hyper-parameters.
